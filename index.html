📋 服务协议 v1.0

PM｜个人管理 是一款面向 Windows 的本地优先个人信息与事务管理软件。使用本软件前，请确认你理解并同意本协议。

一、软件用途

PM 用于个人自我管理、资料整理、提醒、报销记录、社会关系、网站及应用信息和本地资料库管理。软件不应被用于违法、侵权、破坏他人权益或规避安全限制的行为。

二、账户与登录

当前版本提供本地账户注册、登录、邮箱验证码和密码重置能力。账户信息主要用于本机身份识别、敏感信息查看验证和后续联网功能的授权边界准备。用户应自行保管账户密码和邮箱验证码。

三、本地资料库与安全仓

用户资料默认保存在当前 Windows 用户环境下的本机安全仓中。资料库、附件、导出包和备份文件由用户自行管理；用户主动选择连接已有资料库时，软件按原位置连接，不默认复制为新库。

四、敏感信息查看

网站及应用密码默认掩码显示。查看密码时需要输入当前账户密码；未登录或尚未注册账户时，测试默认密码为 88888888。用户应避免在不可信环境中展示或传播敏感密码。

五、授权边界

软件提供云端密文备份与同步授权、AI解析授权、公开协作授权三类隐私选项。除非对应能力完成接入且用户明确授权，软件不会据此自动上传资料、调用 AI 或公开协作。

六、导入、导出与更新

导入、导出、资料库切换、资料库连接、附件导出和 Excel 导出均由用户主动发起。用户应在升级、迁移或连接旧资料库前保留必要备份。

七、协议更新

如软件新增云同步、联网账户、AI服务、团队协作或其他涉及联网处理的能力，本协议将随功能同步更新，并在软件或公开页面中提供查看入口。

🔒 隐私政策 v1.0

PM｜个人管理 是一款面向 Windows 的本地优先个人信息与事务管理软件。本隐私政策说明软件处理用户信息的范围、保存方式、授权边界及用户可使用的数据控制方式。

一、信息处理范围

PM 仅处理用户主动录入、导入或在软件中维护的信息，包括自我管理法则与打卡记录、个人信息结构与资料、报销字段与清单、社会关系字段与资料、网站及应用分类与一般信息、网站及应用密码、缺点与对策框架、提醒事项、附件及用户自行设置的资料库内容。

二、本地存储

PM 采用本地优先方式运行。用户资料默认保存在当前 Windows 用户的本机环境中，应用程序安装目录与用户资料目录相互独立。资料库可进入本机安全仓保存，关闭应用后会重新锁定。

三、账户与邮箱

当前版本提供本地账户注册、登录、绑定邮箱、邮箱验证码和忘记密码功能。邮箱用于注册验证、登录辅助、找回账户和重要安全提醒。测试版验证码可能在本机界面中显示；正式接入邮件服务后，将按用户操作向绑定邮箱发送验证码。

四、敏感密码

网站及应用密码默认不纳入一般信息授权，并在界面中掩码显示。查看密码时需要当前账户密码；未登录或尚未注册账户时，默认密码为 88888888。软件不会以明文方式展示未请求查看的密码。

五、授权选项

用户可分别设置云端密文备份与同步授权、AI解析授权、公开协作授权。每一类授权可按数据大类选择范围。当前版本保存授权边界；在相关云端、AI或协作能力未完成接入并获得用户确认前，不会自动上传、调用 AI 或公开协作。

六、数据传输

除非用户主动执行导入、导出、打开外部链接、未来启用云端备份同步、AI解析或公开协作功能并明确授权，软件不会将用户录入的资料上传至开发者服务器。访问服务协议、隐私协议等公开链接时，将由浏览器打开对应网页。

七、导入、导出与附件

资料库导入、导出、连接已有资料库、附件导出和 Excel 导出均由用户主动发起。用户应自行妥善保管导出文件、附件、资料库位置及相关密码。

八、用户控制

用户可以在软件内维护、修改或删除个人资料，也可以通过数据库管理导出、删除、切换或连接独立资料库。卸载程序不会主动删除用户本机资料目录；用户可在确认不再需要资料后自行删除相关本地数据。

九、政策更新

如 PM 后续新增云同步、联网账户、AI服务、公开协作、分析服务或其他涉及个人信息传输的功能，本隐私政策将同步更新。
